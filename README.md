---

dlex-cnn 是一个以算法练习为目的而写的深度学习框架，支持网络的训练和测试。

---

# 特点

* 轻量级，无第三方库依赖，框架的设计以算法理解为重心，更方便读者了解每个算法细节。

* 保留优化加速前的算法基本版本，注释有详细的演算过程，可方便对应公式。

### 计划内容

* 根据论文研究情况，持续添加新算法的实现；
